## Intro

- A tree is a recursivly defined data structure
- one node is the root 
- can have sub trees
- Root node: The root node R is the topmost node in the tree. If empty then it means the tree is empty.
- if the root is not null things lower are called sub trees
- a node with no children is a leaf

## types of trees

- General trees - normal trees
- Forests - many disconected trees
- Binary trees - each tree can have at most 2 nodes
- Binary search trees - binary tree but aranged in an order
- Expression trees - binary tree to store algebreic expresions
- Tournament trees - like a bracket

## traversing 

- pre-order 
    - each node 
    - root node
    - traverse left tree
    - then traverse right tree
- in order
    - left sub tree
    - root 
    - right sub tree
- post order 
    - left
    - right
    - root
- lvl order
    - nodes on a lvl then next lvl 
    - breadth first search
